# Solar System Simulation

**Welcome to the Solar System Simulation project!** This project utilizes the Three.js library to create a 3D simulation of the solar system, showcasing the planets' orbits around the sun.

## Getting Started

Follow these instructions to get a copy of the project up and running on your local machine for development and testing purposes.

### Prerequisites

Make sure you have the following installed on your machine:

- [Node.js](https://nodejs.org/) - JavaScript runtime
- [npm](https://www.npmjs.com/) - Node package manager

### Installing

1. Clone the repository to your local machine:

    ```bash
    git clone https://github.com/QuentinGehri/systeme_solaire.git
    ```

2. Navigate to the project directory:

    ```bash
    cd systeme_solaire
    ```

3. Install the project dependencies:

    ```bash
    npm install
    ```

### Running the Simulation

Run the development server and open the project in your default web browser:

```bash
npm start
